
# CSS Fundamentals

CSS (Cascading Style Sheets) é usado para estilizar elementos HTML.

## Seletores Básicos

```css
/* Seletor de elemento */
h1 {
    color: blue;
    font-size: 24px;
}

/* Seletor de classe */
.minha-classe {
    background-color: yellow;
}

/* Seletor de ID */
#meu-id {
    margin: 10px;
}
```

## Box Model

O modelo de caixa CSS consiste em:
- Content (conteúdo)
- Padding (preenchimento)
- Border (borda)
- Margin (margem)
