
# HTML Básico

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web.

## Estrutura Básica

```html
<!DOCTYPE html>
<html>
<head>
    <title>Título da Página</title>
</head>
<body>
    <h1>Meu Primeiro Cabeçalho</h1>
    <p>Meu primeiro parágrafo.</p>
</body>
</html>
```

## Tags Importantes

- `<h1>` a `<h6>`: Cabeçalhos
- `<p>`: Parágrafos
- `<div>`: Divisões
- `<span>`: Elementos inline
- `<a>`: Links
- `<img>`: Imagens
